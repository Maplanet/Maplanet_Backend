## 🎯프로젝트 소개

- 메이플랜드 쩔: 잠쩔, 심쩔 구인구직 서비스
- 메이플랜드 겹사: 익명 뒤에 숨어서 남의 자리를 뺏는 유저들을 박제하고 제 3자에게 의뢰를 맡겨 **인기도 하락** 또는 **똑같이 따라다니면서 겹사**를 할 수 있는 서비스
  </br>
  </br>

### ✨주요기능

- 잠쩔, 심쩔 구인구직 게시글
- 겹사, 인기도 하락 의뢰 게시글
- 공지사항
